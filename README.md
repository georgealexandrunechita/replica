# Práctica HTML/CSS – 3 actividades

Colección de 3 ejercicios sencillos de maquetación con **HTML** y **CSS**:

1. Formulario de contacto con campos básicos y selección de lenguajes y nivel de inglés. [web:6]
2. Tarjeta de presentación con foto, nombre, descripción y botones de acción. [web:10]
3. Panel “Zona de planta” con cabecera y tres zonas (lateral izquierdo, zona central y lateral derecho) usando flexbox.

Abre cada `index.html` en el navegador para ver el resultado o modificar los textos y estilos.
